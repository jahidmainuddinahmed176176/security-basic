# JMETER Server Load Testing

## What This Project Does
Load testing on public APIs to measure performance under concurrent user traffic.

## Test Configuration
| Setting | Value |
|---------|-------|
| Virtual Users | 50 |
| Ramp-up Time | 10 seconds |
| Loops per User | 5 |

## Results Summary
- Total Requests: 594
- Average Response Time: 2860 ms
- Throughput: 0.9 requests/second

## How to Run
1. Open JMeter
2. Load `test-plan.jmx`
3. Click the green play button
